**Superstore Sales Analysis Dashboard**

This project analyses retail sales data from a Superstore dataset, obtained from Kaggle, to identify key 
business trends, customer behaviours, and factors affecting sales and profitability.

Using Microsoft Excel, I cleaned and prepared the dataset, created pivot tables and interactive dashboards, 
and used slicers to allow users to explore sales performance across different regions.

The aim of this project was to demonstrate skills in data cleaning, exploratory data analysis, data 
visualisation, and business-focused reporting.

**AIM's**

The analysis focused on:

- Understanding overall sales and profit performance
- Identifying the highest-performing products and categories
- Comparing regional sales performance
- Creating an interactive dashboard for business insights
- Identifying top customer contributions

**Dataset**

The dataset contains retail transaction data from a fictional Superstore, including information on:

- Orders and dates
- Customer details
- Product categories and sub-categories
- Sales and profit
- Discounts
- Regional information
- Shipping details

Dataset source:
Kaggle - Superstore Sales Dataset

**Microsoft Excel**

- Data cleaning and preparation
- Pivot Tables
- Pivot Charts
- Interactive slicers
- Dashboard creation
- Data visualisation
- KPI reporting

**Data Cleaning Process**

The raw dataset was prepared by:

- Checking for missing values
- Removing duplicate records
- Correcting data formats
- Standardising categories and fields
- Creating calculated metrics for analysis

**Key Insights**

Examples of insights identified through analysis:

- Certain product categories contribute significantly more to overall revenue.
- High sales volume does not always correspond to high profitability.
- Regional differences influence both sales performance and profit margins.
